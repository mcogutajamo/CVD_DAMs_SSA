# Application of Decision Analytic Modelling to Cardiovascular Disease Prevention in Sub-Saharan Africa
This code helps to reproduce the graphs presented in the figures of my paper on the Application of decision analytic modelling to CVD prevention in Sub-Saharan Africa. Any use of this code should cite the publication. Thank you. The paper details are as follows:

Oguta, J. O., Breeze, P., Wambiya, E., Kibe, P., Akoth, C., Otieno, P., & Dodd, P. J. (2025). Application of decision analytic modelling to cardiovascular disease prevention in Sub-Saharan Africa: a systematic review. Communications Medicine, 5(1), 46. https://doi.org/10.1038/s43856-025-00772-3
